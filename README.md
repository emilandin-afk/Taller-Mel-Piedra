# Estudio Mel Piedra · Agenda de Pintura

Aplicación de gestión de agenda para clases de pintura.

## Archivos

|Archivo                   |Descripción                      |
|--------------------------|---------------------------------|
|`agenda-pintura.html`     |App privada de gestión (solo Mel)|
|`disponibles-publico.html`|Página pública para Instagram    |

## Cómo subir a GitHub Pages

1. Crear un repositorio nuevo en GitHub (puede ser privado o público)
1. Subir los dos archivos HTML
1. Ir a **Settings → Pages**
1. En **Source** elegir `Deploy from a branch` → rama `main` → carpeta `/ (root)`
1. Guardar → en unos minutos aparece la URL pública

## URLs resultantes

- Agenda privada: `https://TU_USUARIO.github.io/TU_REPO/agenda-pintura.html`
- Página pública: `https://TU_USUARIO.github.io/TU_REPO/disponibles-publico.html`

La página pública es la que se comparte en Instagram.
